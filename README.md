# Backend Development Projects

Welcome to this repository! This contains multiple Node.js and Express backend assignments focusing on API creation, custom logging middleware, and system architecture.

---

## 🚀 Projects Inside

### 1. Vehicle Maintenance Scheduler (`vehicle_maintenance_scheduler`)
A RESTful API built to manage and schedule vehicle maintenance logs efficiently.
* **Features:** CRUD operations for maintenance records, ID-based tracking, and custom response messages.
* **Tech Stack:** Node.js, Express.js

### 2. Logging Middleware (`logging_middleware`)
A custom middleware implementation designed to log incoming server requests for better debugging and monitoring.
* **Features:** Captures HTTP methods, URLs, timestamps, and request parameters.

### 3. Notification System Design (`notification_system_design.md`)
Documentation and design structure for building a scalable system notification flow.
